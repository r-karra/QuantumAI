## üß© 3. Quantum-Specific Python Concepts

Learn these code patterns ‚Äî they are the *core mechanics* of Python-based quantum simulations.

| Concept | Example / Application |
|----------|-----------------------|
| **Matrix Multiplication (`np.dot`)** | Apply a quantum gate: `state = np.dot(H, state)` |
| **Tensor Product (`np.kron`)** | Combine qubits: `np.kron(qubit1, qubit2)` |
| **Normalization** | Ensure state sums to 1: `state /= np.linalg.norm(state)` |
| **Probability Calculation** | `prob = np.abs(state)**2` |
| **Superposition & Measurement Simulation** | Randomly choose based on probability amplitudes. |
| **Complex State Visualization** | Plot Bloch sphere or probability bars with `matplotlib`. |
